# 🚀 TaskFlow Pro

A modern, professional task management application built with React, Vite, and Tailwind CSS. Designed for developers and productivity enthusiasts who want to organize their tasks efficiently.

**🌐 Live Demo:** https://taskflow-pro-cyan-five.vercel.app/

---

## ✨ Features

- 📊 Interactive Dashboard with real-time stats and progress tracking
- ✅ Full Task Management (Create, Edit, Delete, Complete)
- 🎯 Priority Levels (High, Medium, Low)
- 🏷️ Categories (Work, Personal, Study, Health, Other)
- 📅 Due Dates with overdue indicators
- 🔍 Smart Search and Filtering
- 🎨 Dark / Light Mode
- 💾 LocalStorage persistence
- 📱 Fully Responsive Design
- 🎬 Smooth animations (Framer Motion)
- 🔔 Toast notifications
- ⚡ Multi-page navigation (React Router)

---

## 🛠️ Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Router DOM
- Lucide React (Icons)
- Framer Motion (Animations)
- React Hot Toast (Notifications)
- Context API (State Management)

---

## 📁 Project Structure

- src/components/layout - Sidebar, Navbar, Layout
- src/components/tasks - TaskCard, TaskForm, TaskList, Filters, Stats
- src/components/ui - Button, Modal, Input
- src/pages - Dashboard, AllTasks, Completed, Settings
- src/context - ThemeContext, TaskContext
- src/hooks - useLocalStorage
- src/utils - constants, helpers

## 🚀 Getting Started

### Installation

Clone the repository:

    git clone https://github.com/usmanali0999/taskflow-pro.git

Navigate to project folder:

    cd taskflow-pro

Install dependencies:

    npm install

Run development server:

    npm run dev

Open in browser:

    http://localhost:5173

### Build for Production

    npm run build
